Trouver le nombre atypique dans un tableau
=========================================

**ID**: 1642cd11-e57b-4d23-ba61-2bfc0bea7506  

**Énoncé**:  

On vous donne un tableau contenant des entiers.  
Le tableau est soit entièrement composé d’entiers impairs, soit entièrement composé d’entiers pairs, à l’exception d’un seul entier `N`.  

Écrivez un programme qui prend ce tableau en argument et retourne ce « nombre atypique » `N`.  

**Résultat attendu**: afficher (print) l’entier.  

**Exemples:**  

Exemple 1 :  
Entrée : `[2, 4, 0, 100, 4, 11, 2602, 36]`  
Sortie : `11`  

Exemple 2 :  
Entrée : `[160, 3, 1719, 19, 11, 13, -21]`  
Sortie : `160`
